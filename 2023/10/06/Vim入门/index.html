Vim入门

为什么学习Vim

  • Vim熟练后效率超高
  • 服务器上编辑配置文件、写代码必备Vim（当然也可以Vscode远程登录到服务器）
  • Vim优点
    • 简洁
    • 扩展性强
  • Vim缺点
  • Vim哲学：一切操作都可以使用键盘完成

Neovim

  • 安装Neovim：sudo apt-get install -y neovim

Vim的工作模式

  • 普通模式：用来浏览文件、复制粘贴、删除

    • 光标移动
      上下左右箭头
      hjkl(分别代表上下左右)
      4h表示向上移动4行
      w（word）跳到下一个单词的开头
      b（begining）跳到单词开头
      gg跳到文档首行，对应Home键
      G跳到文档末尾
      Ctrl+D(Page Down)下一页
      Ctrl+U(Page Up)上一页
      f(find)+字母：移动到字母的位置，但是好像只能作用于一行

    • 复制粘贴
      y：复制
      yaw（yank拉 all word）：复制整个单词
      y4j：复制包括当前行在内向下的4行

      p（paste）：粘贴

      xshell
